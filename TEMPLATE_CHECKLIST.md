- [ ] Alterar título/posição no `index.html` 
- [ ] Personalizar descrição pessoal
- [ ] Adicionar foto de perfil em `images/profile.jpg`

### ✅ Contato
- [ ] Email institucional
- [ ] Link do LinkedIn
- [ ] Link do GitHub
- [ ] Informações da instituição
- [ ] ORCID (opcional)
- [ ] Lattes (opcional)

- [ ] Adicionar pesquisas em `data/research.json`
- [ ] Listar publicações em `data/publications.json`
- [ ] Incluir eventos em `data/events.json`
- [ ] Configurar repositórios em `data/repositories.json`

### ✅ Personalização Visual
- [ ] Escolher cores principais (opcional)
- [ ] Personalizar logo/favicon (opcional)
- [ ] Ajustar layout se necessário

### ✅ Integração
- [ ] Configurar GitHub username para repositórios automáticos
- [ ] Configurar Google Analytics (opcional)
- [ ] Adicionar links para redes acadêmicas

### ✅ Deploy
- [ ] Testar localmente
- [ ] Fazer commit das alterações
- [ ] Configurar GitHub Pages
- [ ] Verificar funcionamento online
